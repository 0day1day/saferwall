# Frontend

## Setup:

```bash
# install dependencies
yarn install

# serve with hot reload at localhost:8081 (if using windows)
yarn serve_win_cmd
yarn serve_win_powershell

#serve with hot reload at localhost:8081 (if using linux)
yarn serve_lin

# serve with hot reload at localhost:8081 (production global variables)
yarn prod

# build for production with minification
yarn build

# build for production and view the bundle analyzer report
yarn build:report
```
