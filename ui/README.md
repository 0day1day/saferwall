##### Using `yarn`:

```bash
# install dependencies
yarn install

# serve with hot reload at localhost:8081 (development global variables)
yarn dev

# serve with hot reload at localhost:8081 (production global variables)
yarn prod

# build for production with minification (development global variables)
yarn build_dev

# build for production with minification (production global variables)
yarn build_prod

# build for production and view the bundle analyzer report
yarn build:report
```
